- `conversation.name <- session.sessionTitle`
- `conversation.message <- /sys/ts-chat-messages.records[0].contentText`
- `conversation.time <- session.lastMessageAt / updatedAt / createdAt`

- 分流仅依赖 `isSystemSession`，不改动列表布局结构。
