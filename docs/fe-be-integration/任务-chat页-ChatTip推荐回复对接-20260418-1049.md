## 字段映射
- 请求：
  - `sessionId` <- 路由 `sessionId`
  - `historyCount` <- 固定 `12`
  - `userDraft` <- 输入框当前草稿（trim 后为空则不传）
  - `lastAssistantMessageId` <- 当前会话最后一条 AI 消息 id（仅纯数字 id 才传）
- 响应：
  - `suggestions[]` -> `ChatTipItem[]`（`{ id, text }`）

## 调用时机与回退
- 调用时机：点击输入框灯泡按钮。
- 状态流：
  - 已展开时再次点击 -> 收起，并使当前进行中的请求失效。
  - 未展开时点击 -> 展开，进入 loading，发起推荐请求。
  - 成功 -> 渲染推荐项；空数组 -> 渲染“暂无可用建议”提示项。
  - 失败 -> 渲染“获取建议失败，请稍后重试”提示项。
- 回退策略：
  - 请求并发时只接收最后一次结果（requestId 竞态保护）。
  - 会话切换时重置 tips 状态并使旧请求失效。
