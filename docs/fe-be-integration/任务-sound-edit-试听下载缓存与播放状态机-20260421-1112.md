## 字段与缓存键设计（计划）
- 缓存 Key 由以下字段组成：
  - `voiceProfileId`
  - `voiceId`
  - `previewText`
  - `speed`
  - `pitch`
  - `volume`
- TTL：`24h`（`86400000ms`）

## 风险与回退（计划）
- 风险：当前页面是 Web 试听实现（`window.Audio`），在无浏览器缓存能力时需要内存降级。
- 回退：若缓存层异常，回退到“接口返回 URL 直接播放”路径，确保功能可用。
