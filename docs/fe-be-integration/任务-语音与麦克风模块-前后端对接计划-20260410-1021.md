- `POST /sys/ts-chat-sessions/ai-reply`（支持文本输入后生成语音回复）

## 缺口与咨询（需用户确认后再实施）
- 用途：聊天页“按住说话”语音输入（麦克风录音 -> ASR 文本 -> 走 AI 回复）。
- 建议接口：
  - `POST /sys/ts-chat-sessions/asr-draft`
- 入参建议：
  - `sessionId`(Long, 必填)
  - `audioUrl`(String, 必填) 或 `audioBase64`(String, 可选二选一)
  - `lang`(String, 可选，默认 `zh-CN`)
- 出参建议：
  - `recognizedText`(String)
  - `confidence`(Number)
  - `durationSec`(Integer)
- 前端映射：
  - 麦克风录音结果 -> `recognizedText` -> 聊天输入框草稿
- 调用时机：
  - 用户结束录音后触发 ASR，成功后可一键发送到 `/ts-chat-sessions/ai-reply`
- 回退策略：
  - ASR 失败时保留手动文本输入，不阻断聊天。
