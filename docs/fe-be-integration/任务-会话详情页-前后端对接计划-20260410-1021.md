- 角色列表 <- `TsStoryVo.roleBindings` + `TsRoleVo.roleName/avatarUrl`
- 弹层章节内容 <- `TsStoryChapterVo.chapterTitle/chapterDesc/openingContent/missionTarget`

- 风险：作者公开信息字段在故事详情中可能不完整。

## 调用时机与失败回退（本轮）
- 页面首次加载：
  1) 优先读取路由参数 `storyId/id`；
  2) 若缺失，尝试 `sessionId -> /sys/ts-chat-sessions/detail` 反查 `storyId`；
  3) 再缺失时回退到 `/sys/ts-stories` 第一条故事。
- 主数据请求：并发调用
  - `GET /sys/ts-stories/detail`
  - `GET /sys/ts-story-chapters`
- 角色详情：根据 `roleBindings.roleId` 并发调用 `GET /sys/ts-roles/detail` 补全名称与头像。
- 失败回退：
  - 故事详情失败：保留默认标题/文案并提示错误；
  - 章节失败：弹层展示“暂无章节内容”；
  - 单角色失败：该角色回退为默认头像与“角色{id}”。
